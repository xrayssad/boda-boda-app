# Boda Boda App

Ride-hailing web app (Boda Boda) built with React + Flask + Django + Docker.

## How to run with Docker
docker compose up --build

Access:
- Frontend: http://localhost:5173
- Flask API: http://localhost:5000
- Django API: http://localhost:8000
